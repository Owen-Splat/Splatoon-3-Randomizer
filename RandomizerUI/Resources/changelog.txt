<b>0.3.0</b>
<ul>
<li>Adds support for all game versions</li>
<li>Adds support for randomizing enemies and their sizes</li>
<li>Adds support for randomizing item drops</li>
<li>Adds support for randomizing the hero clothes and armor</li>
<li>Adds support for randomized text</li>
<li>The "Enemy Ink Is Lava" setting as returned</li>
<li>Added the option to choose if weapons are randomized. All new weapons are now added</li>
<li>The "Kettles" option has been renamed to "Levels" for better clarity</li>
<li>The "Beatable" setting has been removed, the randomizer now has proper logic</li>
<li>The randomizer app can now be resized</li>
</ul>
<br></br>

<b>0.2.0</b>
<ul>

<li>Adds the option to randomize fuzzy ooze prices</li>
<li>Adds the option to randomize collectables</li>
<li>Adds the option to randomize hero gear upgrades</li>
<li>Adds seasonal weapons support</li>
<li>Adds an option to choose the output format for console or emulator</li>
<li>The "Enemy Ink Is Lava" setting has been replaced with the "Hero Gear Upgrades" setting</li>    
<li>The "Remove Cutscenes" setting has been replaced with the "Skip Cutscenes" setting</li>
<li>Bosses will now be limited to one per site</li>
<li>The random weapons now display when standing over the kettle in Alterna</li>
<li>Fixed a bug where C-1 would not be vanilla, which made it impossible to advance past the first level</li>
</ul>
